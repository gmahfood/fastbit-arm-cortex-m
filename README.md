**Hardware:** STM32 Nucleo / STM32F407 Discovery (ARM Cortex-M4)

Detailed notes live in the [second-brain vault](https://github.com/gmahfood/second-brain) under `20_Projects/fastbit-embedded/`. This repo is code-only.
